# Backend Scripts

This folder contains utility and operational scripts for the backend.

## Scripts

- **run_demo.py** - Demo orchestration script for running demonstrations
- **start_data_feed.py** - Initializes and starts the data feed service
- **move.py** - Utility script for file operations

## Usage

Run scripts from the backend root directory:

```bash
python scripts/run_demo.py
python scripts/start_data_feed.py
```
